## Day 1 - Level 1: Introduction to AI Agents

### Objectives
- Understand the fundamentals of AI agents
- Learn about the agent-environment interaction paradigm
- Explore different types of AI agents and their applications
- Set up the development environment for building AI agents

### Topics Covered
1. What is an AI Agent?
2. Agent vs. Environment
3. Types of AI Agents:
   - Simple reflex agents
   - Model-based reflex agents
   - Goal-based agents
   - Utility-based agents
   - Learning agents
4. Agent architectures and design principles

### Setup Instructions

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/google-agent-ai-course.git
   cd google-agent-ai-course
   ```

2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Navigate to Day 1 materials:
   ```bash
   cd Level-1
   ```
