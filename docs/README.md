## Stock Prediction app

### WiseTrader: All Neural Network Algorithms in One App

WiseTrader is a comprehensive stock prediction application built with Streamlit, integrating various neural network models and a general stock data viewer. It provides an interactive platform for users to analyze stock performance and explore predictions generated by sophisticated machine learning models.

#### Key Features:
- **Stock Watcher**: A primary page to quickly view real-time or historical stock data for any given ticker symbol.
- **Wise CNN**: A dedicated page for Convolutional Neural Network (CNN) based stock price prediction, utilizing historical Open, High, Low, Close, and Volume data.
- **Wise LSTM**: A dedicated page for Long Short-Term Memory (LSTM) based stock price prediction, focusing on historical closing prices.
- **Interactive Interface**: Easy-to-use Streamlit interface for seamless navigation and data input.

#### How to use:
1.  **Enter Stock Ticker**: On the 'Stock Watcher' page, input a stock ticker (e.g., `NV20.NS`, `AAPL`) to view its immediate historical data.
2.  **Navigate to Prediction Models**: Use the sidebar navigation to select between 'Wise CNN' and 'Wise LSTM' pages for more advanced predictions.
